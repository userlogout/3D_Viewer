DOCUMENTATION FOR 3D_VIEWER

This is a program that allows you to rotate an object in three-dimensional space

MAIN FUNCTIONS:

  • Main part
  • Object rotation X, Y, Z
  • A lot of settings
  • Bonus part 1
  • Bonus part 2
  • Taking screenshots
  • Making image
  • GIF
  • allows you to create GIFs and save them immediately

HOW TO USE:

  • To choose a model, you need to select it
  • For the model to appear on the screen, press the "UPLOAD" button
  • To move, rotate and scale the model, you need to enter a value in the box and click on the "MOVING", "ROTATE", "SCALE", "SETTINGS OF EGDE AND NODES" buttons
  • You can choose the type of projections: parallel, central
  • You can choose the color of background and color of your 3Dmodel
  • You have an opportunity to change display type: Dismiss, circle, square
  • You have an opportunity to change display type: Dismiss, circle, square
  • You can give a name to the project you are uploading

Tips:

  • There may be some problems with GIFS
  • Don't use large files, because it can slow down the system
  • Set correct moving, rotate, and scale values..

Settings:

  • There may be problems with rotation in a circle in the axis, due to the fact that all transformations are carried out with matrices.

BASIC SETTING:

  • Select color
  • Select thickness
  • dotted or solid edge
  • Select vertices color
  • Select vertices size
  • Select vertices shape
  • Select projection type: parallel or central
  • Select background color
  • Choose the button SAVE SETTING will apply choose setting

RECOMENDATION:

  • Maximum thickness or size values 10
  • Don't move the mouse when choosing a value
